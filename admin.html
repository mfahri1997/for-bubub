Setup Penyimpanan Online

Supaya setting tidak hilang saat dibuka di device lain, kita perlu GitHub Gist sebagai penyimpanan. Lakukan sekali saja!

Langkah 1: Buka gist.github.com dan buat Gist baru.
Isi file name: birthday-config.json
Isi konten: {}
Pilih "Create secret gist", lalu klik tombol buat.
Langkah 2: Dari URL Gist kamu (contoh: https://gist.github.com/usermu/abc123...), copy bagian ID-nya (huruf panjang di akhir URL).
Langkah 3: Buat GitHub Token di settings/tokens/new.
Centang scope: gist — lalu klik Generate Token dan copy hasilnya.
⚠️ Token GitHub hanya ditampilkan sekali. Simpan dulu sebelum menutup halaman GitHub.
Token disimpan di browser kamu saja (localStorage), tidak dikirim ke mana pun kecuali ke GitHub.
